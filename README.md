# DSA Bank Management System

A **C++ Bank Management System** implementing **Data Structures and Algorithms (DSA)**. This project demonstrates practical use of **arrays, linked lists, stacks, queues, sorting, and searching algorithms** to manage bank operations efficiently.

## Features
- CRUD operations (Create, Read, Update, Delete) on **arrays and linked lists**
- Deposit and withdrawal transactions
- Account search using **linear and binary search**
- Record sorting using **Bubble Sort, Selection Sort, and other algorithms**
- Console-based application to manage bank operations

## DSA Concepts Implemented
- **Arrays:** Store accounts, perform CRUD, and sorting
- **Linked Lists:** Dynamic account management with CRUD operations
- **Stacks & Queues:** Manage transaction orders
- **Sorting Algorithms:** Bubble Sort, Selection Sort, etc.
- **Searching Algorithms:** Linear and Binary Search
